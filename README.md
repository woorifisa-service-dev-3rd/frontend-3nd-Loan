## 주요 기능
### 투자하기 기능
원하는 곳에 투자할 수 있는 기능입니다.

### 대출상품 추천 기능
본인의 재산과 직업을 입력하면 맞춤형 대출 상품을 제공합니다.

### 원하는 조건의 상품 검색 기능
본인의 원하는 상품을 필터링 기능을 통해 검색할 수 있습니다.
